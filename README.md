# AI Job Readiness Analyzer

An AI-powered web application that analyzes resumes 
against job descriptions using LLaMA 3.3-70B model 
hosted on Groq API.

## Tech Stack
- Python
- Streamlit
- Groq API
- LLaMA 3.3-70B
- PyPDF2
- python-docx

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. Create .env file with GROQ_API_KEY=your_key
4. streamlit run app.py
